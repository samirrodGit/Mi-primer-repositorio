Un iframe en HTML (Inline Frame) es una etiqueta que se utiliza para incorporar o insertar otro documento HTML dentro de la página actual. Este elemento permite embeber contenido de otras fuentes,como documentos HTML, videos, mapas, widgets, o incluso páginas web completas, en una sección específica de una página. Los iframe son útiles para integrar contenido de terceros, como mapas de Google, videos de YouTube, o widgets de redes sociales, en una página sin necesidad de cargar todo el contenido directamente en el HTML principal.
